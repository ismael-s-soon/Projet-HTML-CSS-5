# Huddle Landing Page

## Description
Landing page pour la plateforme fictive **Huddle**. Projet réalisé pour pratiquer **HTML & CSS**, créer une page **moderne, responsive et attrayante** avec navigation, images, icônes et pied de page complet.

## Objectifs
- Structurer une page web avec HTML sémantique  
- Utiliser images, icônes et polices externes  
- Mettre en forme avec CSS et Flexbox  
- Rendre la page responsive  

## Spécifications
- Structure HTML sémantique  
- Polices et couleurs respectées  
- Responsive design et Flexbox  
- Code propre et bien organisé  

## Style Guide
- **Layout :** multi-colonnes desktop → une colonne mobile  
- **Breakpoints :** max-width 769px et 425px  
- **Couleurs :** Pink, Light Pink, Light Red, Dark Cyan, Grayish Blue, White  
- **Typographie :** Headings: Poppins 600, Body: Open Sans 400/700  
- **Icons :** Font Awesome pour réseaux sociaux et contact  
- **Images :** bg-hero-desktop.svg / bg-hero-mobile.svg
